Verifier les mises a jour

Procedure recommandee :
  1. Allez dans Sauvegardes et creez une sauvegarde de la base de donnees
  2. Cliquez sur "Verifier les mises a jour" pour consulter la derniere version disponible
  3. Si une mise a jour est disponible, cliquez sur "Lancer la mise a jour"
  4. Redemarrez l'application apres la mise a jour
  5. Allez dans Administration pour appliquer les eventuelles migrations de base de donnees

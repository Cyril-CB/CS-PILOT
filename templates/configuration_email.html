Configuration Gmail :
Pour utiliser Gmail comme service d'envoi, vous devez creer un mot de passe d'application dans votre compte Google.
Le mot de passe habituel de votre compte Gmail ne fonctionnera pas.

Etapes :
1. Connectez-vous a votre compte Google sur myaccount.google.com
2. Allez dans Securite > Verification en 2 etapes (doit etre activee)
3. En bas de la page, cliquez sur Mots de passe des applications
4. Creez un mot de passe pour "CS-PILOT" et copiez le code a 16 caracteres
5. Utilisez ce code ci-dessous comme mot de passe SMTP

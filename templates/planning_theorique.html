{% if plannings %}Modifier{% else %}Créer{% endif %} un planning

Planning alterné : Si vous avez des horaires qui changent chaque semaine (ex: semaine 1 = 35h, semaine 2 = 28h), créez un planning pour chaque semaine.

Important : Choisissez un LUNDI qui sera considéré comme le début d'une "semaine 1". L'alternance se calculera automatiquement à partir de cette date.
Exemple : Si vous choisissez le lundi 06/01/2026, alors :
• Semaine du 06/01 = Semaine 1
• Semaine du 13/01 = Semaine 2
• Semaine du 20/01 = Semaine 1
• ...et ainsi de suite

Important : Ce planning sera utilisé pour tous les calculs à partir de cette date. Si vous modifiez un planning existant, saisissez la date à partir de laquelle le nouveau planning s'applique.

Conseil : Laissez les horaires vides pour les jours non travaillés. Les heures vides = jour de repos.

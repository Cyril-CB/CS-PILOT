💡 Bonnes pratiques

  • Sauvegardez regulierement : avant chaque mise a jour, en fin de mois, ou apres des modifications importantes
  • Telechargez vos sauvegardes : conservez une copie sur un autre support (cle USB, disque externe, cloud)
  • Documents uploades : les archives ZIP conservent l'arborescence de vos sous-dossiers
  • Restauration : une sauvegarde de securite est creee automatiquement avant toute restauration
  • Rotation automatique : les 20 dernieres sauvegardes sont conservees, les plus anciennes sont supprimees automatiquement

Ligne de commande : Vous pouvez aussi utiliser python backup_db.py pour sauvegarder, python backup_db.py --list pour lister, et python backup_db.py --restore <fichier> pour restaurer.
